# Predictive Maintenance Analytics (Industrial IoT)

## Overview
This project analyzes industrial IoT sensor and maintenance data to identify failure risks, optimize maintenance planning, and reduce downtime.

## Business Objectives
- Identify high-risk machines
- Understand failure drivers
- Optimize maintenance scheduling
- Reduce repair costs and downtime

## Tech Stack
- SQL (PostgreSQL)
- Python (pandas, matplotlib)
- Excel (data validation)
- Power BI (French dashboard)

## Key KPIs
- Failure Rate
- Remaining Useful Life (RUL)
- Repair Cost
- Reliability by machine/type

## Project Structure
- data/raw: raw datasets (immutable)
- data/processed: cleaned datasets
- data/mart: analytics-ready datasets
- notebooks: exploration + production notebooks
- src: reusable python modules
- sql: DDL + KPI queries + mart views
- powerbi: Power BI dashboard (FR)
- reports: executive summary (FR)

## 🇫🇷 Résumé
Projet d’analyse de maintenance prédictive utilisant Python, SQL, Excel et Power BI pour identifier les machines à risque, analyser les facteurs de panne et optimiser la maintenance.
